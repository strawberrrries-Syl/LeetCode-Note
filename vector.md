## 1. insert
```cpp
vec.insert(pos,elem); // insert new element at position.

// How to use:
auto it = vec.insert(vec.bigin(), 1);   // insert 1 on position 0

```

## 2. erase()
```cpp
vec.erase(pos); // delete element at position. size - 1;
// or
vec.pop_back(); // as stack;
```
